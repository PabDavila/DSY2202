# Kyrius Garage

Aplicación Angular de venta de modelos a escala, desarrollada como parte del curso **Desarrollo Fullstack II (DSY2202)**.

## Tecnologías usadas

- Angular 17+
- TypeScript
- Bootstrap 5
- Docker + NGINX
- Formularios Reactivos
- Pruebas unitarias con Karma & Jasmine
